# 2ColumnSite: Tachikawa City Website

## プロジェクト概要 (Project Overview)
このウェブサイトは 2 カラムレイアウトを用いた東京都西部に位置する立川市を紹介するサイトです。サイドバーナビゲーションと広々としたメインコンテンツエリアによる 2 カラム構造を特徴としています。サイト内のすべての写真は制作者自身が撮影したオリジナル写真です。

## 技術的特徴
- **レスポンシブデザイン**: モバイルからデスクトップまで様々な画面サイズに適応
- **モダンなレイアウト**: 左側に固定サイドバー、右側に流動的なコンテンツエリア
- **アニメーション効果**: スライドショー、スクロールアニメーションなどの動的要素
- **インタラクティブ要素**: ギャラリービュー、画像拡大機能、ナビゲーションなど
- **Tailwind CSS**: スタイリングに使用
- **日本語 UI**: 全体的に日本語インターフェースを採用

## サイト構造
- **トップセクション**: 立川市の代表的な風景のスライドショー
- **紹介セクション**: 立川市の概要説明
- **魅力セクション**: 立川の主な観光スポットやハイライト
- **飲食セクション**: 立川市のレストランや食文化の紹介
- **交通セクション**: 立川市へのアクセス方法と市内交通の案内
- **ギャラリーセクション**: 立川市の様々な風景や施設の写真集

## モバイル対応
モバイルデバイス向けにハンバーガーメニューを実装し、小さい画面でも使いやすいナビゲーション体験を提供しています。コンテンツもモバイル画面に最適化されるようレイアウトが調整されます。

## 特徴的な機能
- ナビゲーションリンクによる各セクションへのスムーズなスクロール
- 画像ギャラリーの閲覧・ページング機能
- 画像クリックによる拡大表示モーダル
- トップへ戻るボタン
- アニメーションによるビジュアルエンゲージメント

このサイトは 2 カラムレイアウトの実践的な応用例として、視覚的に魅力的なコンテンツと機能的なナビゲーションを両立させています。
